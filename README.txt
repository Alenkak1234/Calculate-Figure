Модуль дл рассчета плозади фигуры. Реализован расчет площади треугольника по трем сторонами и круга по радиусу.

Пример использования модуля:

fig = input('Введите тип фигуры(треугольник, круг...)\n')
figure = Figure(fig)
print(figure.calculate_square())
